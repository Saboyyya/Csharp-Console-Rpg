# Csharp-Console-Rpg
#  Mini RPG em C#

Mini RPG desenvolvido em C# com foco em **lógica de programação**, **Programação Orientada a Objetos (POO)** e **mecânicas básicas de jogos**, rodando no **terminal**.

Este projeto foi criado com o objetivo de estudo e evolução contínua, podendo futuramente ganhar uma versão gráfica.

---

## Funcionalidades

- Sistema de batalha em turnos
- Ataque com chance de crítico
- Inimigos diferentes (Goblin, Orc)
- Sistema de vida (HP)
- Escolhas do jogador via menu no terminal
- Uso de herança e polimorfismo

---

##  Tecnologias Utilizadas

- C#
- .NET
- Programação Orientada a Objetos
- Console Application
